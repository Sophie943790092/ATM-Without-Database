# ATM-Without-Database()
项目功能描述:实现银行ATM项目(不使用数据库)--后续会逐步完善
实现银行ATM的存款,取款,转账,查询
在取款金额大于10000时激活事件BigMoneyFetched (一大笔钱被取走了）,弹出”一大笔钱被取走了”的警告.
定义一个异常类BadCashException,表示有坏的钞票,实验指导书上要求有三个之一的概率时就抛出,故使用Random随机函数进行随机数的设置.
